📊 Global Tech Layoffs Analysis (2020-2025)
📌 Project Overview
This project involves a comprehensive analysis of global tech layoffs spanning from 2020 to 2025. By merging three distinct datasets, I developed an end-to-end data pipeline to uncover layoff trends, the impact of company funding, and geographical hotspots across the tech industry.

🛠️ Tech Stack Used
Language: Python

Libraries: * Pandas: Data manipulation and merging.

NumPy: Numerical operations and handling missing data.

Matplotlib & Seaborn: Advanced data visualization and heatmaps.

Tools: VS Code

🧹 Key Steps Performed
Data Merging: Consolidated data from 3 different CSV sources, aligning disparate column names (e.g., 'usstate', 'USState', 'location_HQ') into a unified schema.

Data Cleaning: Cleaned and pre-processed over 4,100+ rows of data. Handled significant missing values (NaN) in layoffs counts, funding amounts, and company stages.

Exploratory Data Analysis (EDA):

Identified the Top 10 Companies and Industries most affected by layoffs.

Analyzed Yearly Trends to identify peaks in the layoff cycle.

Examined the Correlation between "Money Raised" (Funding) and the scale of layoffs.

Automated Reporting: Developed a script to generate a multi-page PDF report containing 7 distinct visualizations, including bar charts, boxplots, scatter plots, and heatmaps.

📈 Key Insights
Peak Year: 2023 was identified as the peak year for global tech layoffs.

Industry Impact: The Consumer and Retail sectors were the most heavily impacted industries.

Funding Paradox: The analysis revealed that high funding does not necessarily prevent layoffs; several well-funded startups conducted massive staff reductions.

Geographical Hotspots: Visualized major layoff clusters in tech hubs like Silicon Valley and major Indian cities.
